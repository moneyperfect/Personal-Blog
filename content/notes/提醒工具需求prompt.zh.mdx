# My Reminder - Project Summary & Requirements

## 1. Project Overview
**My Reminder** is a mobile application built with **React Native (Expo)** designed to help users track tasks with specific expiration dates and times. The application emphasizes a "Warm Tech" aesthetic, focusing on tactile feedback, clean card-based UI, and smooth micro-interactions.

## 2. Tech Stack
- **Core Framework**: React Native (Expo SDK 54)
- **Language**: TypeScript
- **Styling**: NativeWind (Tailwind CSS for React Native)
- **State Management**: React `useState` / `useEffect`
- **Persistence**: `@react-native-async-storage/async-storage` (Local Data Persistence)
- **Animations**: `moti` (powered by `react-native-reanimated`)
- **Icons**: `lucide-react-native`
- **Date Handling**: `@react-native-community/datetimepicker`
- **UX Feedback**: `expo-haptics`

## 3. Functional Requirements

### 3.1 Core Features
1.  **Add Reminder**:
    *   Input text field for task description (supports multiline).
    *   Date Picker: Select expiration date.
    *   Time Picker: Select expiration time.
    *   Validation: Prevent adding empty reminders.
2.  **List View**:
    *   Display a list of active reminders.
    *   **Sorting**: Automatically sort items by expiration time (soonest first).
    *   **Expired State**: Visually distinguish expired items (Red accent, strikethrough text).
3.  **Persistence**:
    *   Save data locally using `AsyncStorage`.
    *   Reload data automatically upon app launch.
4.  **Task Completion**:
    *   Delete/Complete button to remove items from the list.

### 3.2 User Experience (UX)
*   **Haptic Feedback**: Trigger vibration feedback on adding items (Success) and deleting items (Impact).
*   **Animations**:
    *   Entry animations for list items (Fade In + Slide Up).
    *   Smooth transitions for input fields.
    *   Press interactions (Scale down on tap).

## 4. Design Guidelines ("Warm Tech")
*   **Color Palette**:
    *   Background: `bg-slate-50` (Off-white/Paper feel).
    *   Primary Accent: `amber-400` (Warm Yellow/Orange).
    *   Secondary/Success: `emerald-500`.
    *   Error/Expired: `red-400` / `red-50`.
*   **Typography**: Clean sans-serif, using `zinc-900` for primary text and `zinc-500` for secondary text.
*   **Shapes**:
    *   Cards: `rounded-2xl` or `rounded-3xl`.
    *   Buttons: `rounded-full`.
*   **Visuals**: Use shadows (`shadow-sm`, `shadow-md`) to create depth.

## 5. Data Structure
**Interface `Reminder`**:
```typescript
interface Reminder {
  id: string;        // Unique ID (random string)
  text: string;      // Task description
  expiresAt: string; // ISO 8601 Date String
}
```

## 6. Project Structure (Key Files)
- `App.tsx`: Main entry point containing UI logic, state, and styles.
- `babel.config.js`: Configured for `nativewind/babel` and `react-native-reanimated/plugin`.
- `tailwind.config.js`: Custom theme configuration extending colors.
- `package.json`: Dependency management.

## 7. Installation & Run
```bash
# Install dependencies
npm install

# Start the development server
npm start
```
