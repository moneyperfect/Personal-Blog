# 第一学期数学期中考试复盘

## 📊 状态评估
**心力耗竭 (Exam Fatigue)**

## 🔍 现场复原 (The Moment)
- **时间点**：考试仅剩最后10分钟。
- **已完成**：所有已知题目均已答完。
- **状态**：处于完全力竭状态。
- **外部条件**：
    - 手机里有现成的答案。
    - 监考环境为“两松一紧”，存在明显的套利空间。
- **理性判断**：这是一个高期望值（EV）的冒险，收益远大于风险。
- **执行失败过程**：
    1.  **生理反应**：试图执行时，遭遇严重身心解离。表现为极度紧张、胸闷气短、手脚僵硬不听使唤。
    2.  **心理干预失败**：试图使用“闭左眼”仪式进入俯瞰模式，但瞬间被本能（搜索威胁）强行睁眼打断，无法建立观察者视角。
    3.  **代偿行为**：为缓解焦虑，进行了无意义的翻卷子动作，浪费了最后的决策窗口期。
- **结果**：既未能执行抄答案的策略，也未能通过理性安抚自己，在混乱中结束考试。

## 🧠 深度复盘 (Root Cause Analysis)
1.  **杏仁核劫持 (Amygdala Hijack)**
    - 在极度疲劳和高压下，大脑的原始“安保系统”（杏仁核）权限高于理性“CEO”（前额叶）。
    - 身体将“被抓风险”误判为生存威胁，直接切断了理性的决策与执行路径。
2.  **能量枯竭**
    - 长时间的答题已耗尽意志力（葡萄糖与神经递质）储备。
    - 最后阶段，油箱已空，无法启动需要高能耗的深度思考与情绪控制功能。
3.  **工具失效**
    - “闭左眼”作为心理暗示开关，其信号强度在高强度的生理恐惧面前过于微弱。
    - 类比：试图用一杯水去熄灭一车着火的柴薪。

## 🚀 下一步行动 (Action Items)
1.  **重启生理耐受训练**
    - **行动**：每日进行冷水训练。
    - **目的**：提高身体对突发压力的耐受阈值，模拟在考场窒息感中保持呼吸与冷静的能力。
2.  **引入强物理干预**
    - **情境**：下次再遇身体僵硬、被恐惧“冻住”的情况。
    - **方法**：不使用纯心理暗示，立即采用：
        - **痛觉锚点**：用力掐手心。
        - **战术呼吸**：采用4-4-4（吸气4秒，屏息4秒，呼气4秒）呼吸法。
    - **目的**：通过强烈物理信号，强制打断杏仁核的劫持广播，夺回身体控制权。
3.  **接受并执行兜底策略**
    - **认知**：承认在精力耗尽状态下，无法进行精密的高风险操作。
    - **行动**：若判断处于此状态，则主动放弃高冒险计划，转而执行低风险的兜底策略（如仔细检查答案、对剩余题目进行合理猜测）。
    - **目的**：避免因“想做却做不到”而产生的二次内耗，确保最低收益。
